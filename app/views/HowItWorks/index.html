How It works

Many other sites make you really work to find the tab you want. We have tried to make it as easy as possible.

  • Simply search for the song, artist, or album you are interested in
  • Decide whether you want to learn the chords or tablature
  • Start playing...

We have set up wiki style editing for each tab, so if there's something not quite right, click the edit button and make the change
